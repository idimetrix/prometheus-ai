## What is Prometheus?

Prometheus is an AI-powered engineering platform that uses 12 specialist agents to take your project from requirements to production deployment. You describe what you want built, and Prometheus handles architecture, implementation, testing, and deployment.

## Key Concepts

**Sessions** - A session represents a single interaction with Prometheus. You provide a prompt describing your task, and Prometheus orchestrates the appropriate agents to complete it.

**Agents** - Prometheus uses 12 specialist agents, each with a specific role:

- **Orchestrator** - Coordinates all other agents
- **Discovery** - Analyzes requirements and gathers context
- **Architect** - Designs system architecture
- **Planner** - Creates detailed implementation plans
- **Project Brain** - Maintains project knowledge and context
- **Frontend Coder** - Implements UI components
- **Backend Coder** - Implements server-side logic
- **Integration Coder** - Handles integrations and APIs
- **Test Engineer** - Writes and runs tests
- **CI Loop** - Manages continuous integration
- **Security Auditor** - Reviews code for vulnerabilities
- **Deploy Engineer** - Handles deployment and infrastructure

**Modes** - Five interaction modes:

- **Task** - Execute a specific coding task
- **Ask** - Ask questions about your codebase
- **Plan** - Generate a detailed plan without executing
- **Watch** - Monitor and auto-fix CI/CD failures
- **Fleet** - Deploy multiple agents in parallel

## Credits

Prometheus uses a credit system for billing. Each operation consumes credits based on the complexity and resources used. Different plan tiers include different monthly credit allocations.
